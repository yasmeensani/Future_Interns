# 📊 Customer Retention & Churn Analysis

## 📌 Project Overview
This project analyzes customer behavior data to identify churn patterns, retention trends, and customer lifetime value insights for a subscription-based business. The goal is to understand why customers leave, what drives retention, and how the business can improve customer loyalty using data-driven strategies.



## 🎯 Objectives
- Identify customer churn patterns  
- Analyze key drivers of customer retention  
- Perform cohort analysis to track customer behavior over time  
- Evaluate customer lifetime trends and value  
- Generate actionable insights to reduce customer loss  



## 🛠️ Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Seaborn)  
- Power BI (for interactive retention dashboards)  
- Excel (for quick cohort and churn calculations)  



## 📂 Dataset Description
The dataset contains subscription-based customer records with fields such as:

- Customer ID  
- Signup Date  
- Subscription Plan  
- Last Activity Date  
- Churn Status (Yes/No)  
- Monthly Charges  
- Tenure (months)  
- Usage Frequency (if available)  



## 🧹 Data Cleaning & Preparation
- Handled missing values and duplicates  
- Converted date fields into datetime format  
- Created churn labels (Active vs Churned customers)  
- Standardized subscription categories  
- Calculated customer tenure and lifetime metrics  



## 📊 Key Metrics
- 📉 Churn Rate (%)  
- 📈 Retention Rate (%)  
- 👥 Customer Lifetime Value (CLV)  
- ⏳ Average Customer Tenure  
- 📊 Monthly Active Users (MAU)  
- 🔁 Cohort Retention Rates  



## 📈 Analysis Performed

### 🔹 Cohort Analysis
- Grouped customers by signup month  
- Tracked retention over time  
- Identified drop-off patterns across cohorts  

### 🔹 Churn Analysis
- Analyzed churned vs retained customers  
- Identified high-risk segments  
- Compared behavior patterns of churned users  

### 🔹 Customer Lifetime Analysis
- Estimated average customer lifespan  
- Evaluated revenue contribution per customer segment  



## 📊 Dashboard Features
- Monthly churn trend visualization  
- Cohort retention heatmap  
- Customer segmentation (high vs low value)  
- Subscription plan performance  
- Churn risk analysis charts  



## 🔍 Key Insights
- Most churn occurs in early customer lifecycle (first 1–3 months)  
- Certain subscription plans have higher retention rates  
- Low engagement users are more likely to churn  
- Cohorts acquired during promotions show weaker long-term retention  



## 🚀 Recommendations
- Improve onboarding experience to reduce early churn  
- Introduce loyalty rewards for long-term users  
- Target at-risk customers with personalized offers  
- Optimize underperforming subscription plans  
- Increase engagement through reminders and feature adoption campaigns

- Cohort Analysis  
- Customer Lifetime Value (CLV) Modeling  
- Data Visualization & Business Intelligence  


## 📌 Outcome
This project provides a clear understanding of customer behavior patterns and delivers actionable strategies to improve retention and reduce churn in subscription-based businesses.
